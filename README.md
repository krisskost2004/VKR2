# Сравнительный анализ метаэвристических алгоритмов оптимизации для настройки регуляторов

Данный проект реализован в рамках выпускной квалификационной работы бакалавра по направлению 27.03.04 «Управление в технических системах». 
Цель работы — сравнение эффективности пяти современных метаэвристических алгоритмов при решении задач параметрической настройки регуляторов для технических систем с различной динамикой.

## Алгоритмы

В проекте реализованы и протестированы следующие алгоритмы:

| Алгоритм | Аббревиатура | Тип |
|----------|---------------|------|
| Метод роя частиц | PSO | Роевой |
| Алгоритм серого волка | GWO | Иерархический |
| Алгоритм китовой оптимизации | WOA | Биомиметический |
| Алгоритм ястребов Харриса | HHO | Кооперативный |
| Алгоритм слизевидных амёб | SMA | Биологически инспирированный |

Все алгоритмы реализованы с единым интерфейсом (`BaseOptimizer`) и поддерживают:
- задание границ поиска,
- настраиваемые размер популяции и число итераций,
- запись истории сходимости (с начальной оценки популяции),
- воспроизводимость через `seed`.

## Объекты управления

Исследование проведено на трёх моделях, охватывающих основные классы динамических систем:

1. **Линейная минимально-фазовая система** – двигатель постоянного тока с ПИД-регулятором.
   - Передаточная функция двигателя:  
   - Целевая функция: ITAE + штраф за перерегулирование + штраф за ошибку на «хвосте».

2. **Неустойчивая система** – перевёрнутый маятник на тележке.
   - Линеаризованная модель в пространстве состояний.
   - Закон управления: \( u = -Kx \).

3. **Нелинейная система с шумами и ограничениями** – два связанных резервуара.
   - Нелинейные потоки, квадратичная зависимость сопротивления от уровня.
   - Динамически изменяющиеся уставки.
   - Гауссов шум измерений (фиксируется через `seed` для воспроизводимости).
   - Целевая функция учитывает абсолютную ошибку, плавность управления, колебания и финальную ошибку.
