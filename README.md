# Исследование современных оптимизационных алгоритмов в задачах управления техническими системами
В работе исследуются пять метаэвристических алгоритмов на трёх классических задачах теории управления. Основное внимание уделяется:
- **Точности** найденных решений (значение целевой функции)
- **Вычислительной эффективности** (время выполнения, количество вызовов функции)
- **Стабильности** (стандартное отклонение результатов при множественных запусках)
- ## Установка и запуск

### Предварительные требования
- **Python 3.8**
- **Git**

### Необходимые библиотеки
numpy, scipy, matplotlib, pandas, control, psutil, но также зависимости прописала в файле requirements.txt
# Установить все зависимости:
pip install -r requirements.txt
# Запустить эксперименты
python experiment.py
После успешного запуска в папке experiment_results/ появятся:
summary_results.csv — сводная таблица с результатами
results.json — детальные результаты всех запусков
convergence.json — данные для графиков сходимости

В папке plots/ сгенерируются все графики:
convergence_*.png — графики сходимости
quality_speed_comparison.png — сравнение качества и скорости
radar_chart.png — радарная диаграмма
